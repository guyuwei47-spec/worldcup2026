② 算法 · 三步走

Step 1 · 计算"队伍强度"
strength = ((elo - 1400) / 16 + 55) × 0.6
         + (attack + defense + form) / 3 × 0.4

Elo 占 60%（客观主导），主观评估占 40%（捕捉短期状态）。这是 Sprint 2.1 改的关键决定——之前主观为主导致排行榜里"摩洛哥/挪威"等 Elo 不够但主观虚高的队排进 Top 10，违反诚信。改完之后西班牙凭 Elo 2157 升至 #1，符合一线博彩公司排序。

Step 2 · 推导期望进球数 λ（泊松分布的均值）
strengthDiff = strengthA - strengthB
λA = base + strengthDiff × scale + homeAdv  // 强队进球期望更高
λB = base - strengthDiff × scale            // 弱队进球期望更低

两支队的"期望进球"不是猜的，而是从强度差按比例推导。强队 vs 弱队 λ 差距大；势均力敌 λA ≈ λB ≈ 1.2。

Step 3 · 用独立泊松联合分布枚举所有比分（这是 Sprint 2.3 的核心改进）
for a in 0..7:
  for b in 0..7:
    P(a-b) = pois(a, λA) × pois(b, λB)   // 独立泊松联合分布
排序取概率最高的 3 个 → Top-3 比分 predictedScore = Top-1.score // 主比分与 Top-1 对齐

为什么不用 Skellam？Skellam 给的是"进球差 = -3, -2, ..., +3"分布，把"几比几"信息抹掉了。我们要展示具体比分，所以用联合分布枚举 8×8=64 个比分。
为什么 predictedScore = Top-1？之前用 Math.round(λ) 得到的"边缘均值"会和联合分布最大比分打架（同一张卡片出现两个相互矛盾的"主比分"）。对齐后，加拿大 vs 波黑 从 2-1 变 1-1，与实际平局结果一致。

③ 已知局限 · AI 会在哪里翻车

这一节比"我们多牛"更重要——好球迷站应该敢承认自己哪里弱。当前 10 场小组赛战绩：方向对 4/10 (40%)，略高于纯随机 33% 但仍远不够说明算法质量（10 场样本依然不够）。原因有三：

⚠️ 局限 1 · Elo 历史滞后盲区

Elo 反映的是长期累积实力，对"近期归化爆发"或"主力老化"反应慢。典型翻车 1：6/13 美国 4-1 巴拉圭——AI 因巴拉圭历史 Elo 较高预测它赢 (2-0)，但实际美国近期归化潮 + 巴拉圭主力老化让强弱关系完全反转。典型翻车 2：6/14 荷兰 2-2 日本——AI 押荷兰 1-0 胜，但日本旅欧球员近 4 年成长 + 荷兰后防失误终止历史悬殊。解药：未来引入 H2H 近期交锋、伤病数据；目前只能靠主观评估打补丁。

⚠️ 局限 2 · 平局收敛偏差

独立泊松联合分布在 λA ≈ λB ≈ 1（小组赛常态）时，1-1 / 0-0 / 1-0 / 0-1 这几个低比分会瓜分大部分概率，让"势均力敌"的比赛系统性预测为 1-1。这数学上是对的，但作为"主比分"展示给球迷会显得保守。
比分层解药（Sprint 3.1 已实施 ✅）：当 Top-1 是平局且一方明显占优时，让 predictedScore 退回 Top-2；Top-3 列表保持真概率不变。
概率层解药（Sprint 7.3 已实施 ✅）：用完整 72 场小组赛校准对称胜/平/负模型，势均力敌时允许平局成为首选。更新后方向命中由 43/72 提升至 46/72，RPS 从 0.2096 降至 0.1889，并在 20 场实际平局中预测 9 场、命中 5 场。仍需更多赛事做跨届验证，不能把同届回测当成未来保证。

⚠️ 局限 3 · 小样本无统计意义

10 场样本已具备初步复盘价值，但仍不足以证明模型能跨赛事稳定（一般认为 100 场以上更稳）。看战绩看板时请同时看「对照基准」——AI 必须长期跑赢"33% 纯随机"和"永远选强队"才算合格。当前 Top-3 命中率 6/10 = 60%，会随完整赛果动态重算。

④ 五条铁律（自我监督）

  1. 所有展示数据必须真实——禁止合成、随机游走、占位符代替真实比分。
  2. 客观与主观分离——主观字段必须有 ※ 角标 + tooltip 声明。
  3. 排行榜必须用 getTeamStrength()——不能拿单一主观字段（如纯 attack）排序，否则会让虚高队伍冒头。
  4. AI 也会错——战绩看板逐场展示对错，错了就标 ❌ 翻车，不藏拙不洗。
  5. 所有预测确定性可复现——同一数据版本下刷新页面结果不变，禁止 Math.random()。

⑤ 怎么读懂一张预测卡片

  • 主比分（如 2-1）：联合分布的 Top-1，即"AI 认为最可能的具体比分"。
  • 一句话解读：基于 probDiff / λ总和 / tier 落差自动套 7 个场景模板（强强对话 / 爆冷温床 / 实力悬殊 / 闷战 / 进球大战 / 防守战 / 优势局），完全确定性。
  • 绿色胶囊 (Top-1)：最可能比分及其概率（一位小数 %）。
  • 金色胶囊 (Top-2)：第二可能比分。
  • 灰色胶囊 (Top-3)：第三可能比分。三者概率和约 30%~55%（剩下 60% 散在其它比分，符合足球真实分散度）。
  • key-factors：算法给出的 3-5 条"为什么这么预测"的理由，便于球迷判断 AI 是否考虑周全。
